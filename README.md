# Expense-Tracker-Dashboard-using-Python-Streamlit
Description:
An interactive Expense Tracker web app built using Python and Streamlit. Users can add, manage, and analyze daily expenses with category-wise tracking and visual charts. The dashboard provides insights through graphs and summaries to help monitor spending habits and improve personal budgeting.

Folder-Structure
expense-tracker/
│
├── app.py
├── requirements.txt
├── README.md
└── Output images

## 🛠 Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib
- SQLite

- ## ▶️ How to Run
1. Clone the repository
2. Install requirements
   pip install -r requirements.txt
3. Run the app in Terminal 
   streamlit run expense_tracker.py
